# 🚀 DHRUVA — AI-Powered Study Planner

> A smart AI study companion that creates personalized learning plans, tracks progress, and improves study efficiency.

---

## 📌 Problem Statement

Students often struggle with:
- Poor time management
- Inconsistent study routines
- Overwhelming schedules
- Lack of personalized study planning

Traditional planners are static and fail to adapt to a student’s goals, workload, and productivity patterns.

---

## 💡 Solution

DHRUVA is an AI-powered study planner that intelligently generates personalized study schedules, manages tasks, and helps students stay productive using adaptive planning and analytics.

---

## ✨ Features

- 🤖 AI-generated personalized study plans
- 📅 Smart scheduling system
- ✅ Task & goal management
- 📈 Productivity tracking
- 🎯 Adaptive recommendations
- 🌙 Modern responsive UI
- ⚡ Fast and interactive experience

---

## 🛠️ Tech Stack

### Frontend
- React.js / Vite
- Tailwind CSS
- JavaScript

### Backend
- FastAPI
- Python

### AI Integration
- Groq API
- LLM-powered study planning

---

## 🧠 How It Works

1. Users enter subjects, study goals, and available time.
2. AI analyzes workload and priorities.
3. DHRUVA generates an optimized study schedule.
4. Users track progress and improve consistency over time.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/divakarpathak-code/DHRUVA.git
```

---

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```
